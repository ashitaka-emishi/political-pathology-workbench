# Standards Alignment

The spike keeps tooling lightweight while preserving compatibility with established research-data standards.

- FAIR for data principles
- Frictionless Data Package for future dataset bundles
- CSL JSON for bibliographic source metadata
- Dublin Core / DataCite for metadata crosswalks
- PROV-O / PAV-inspired provenance
- Toulmin model for argument roles
- W3C Web Annotation-inspired passage annotation
- SKOS-inspired controlled vocabulary
- CIDOC CRM-inspired historical modeling
- TEI-compatible future source text encoding
- JSON Schema Draft 2020-12 for validation
- PRISMA-inspired literature review tracking
- MIPVU alignment for metaphor analysis
- Cohen's kappa / Krippendorff's alpha for future reliability work
- ADRs for architecture governance
