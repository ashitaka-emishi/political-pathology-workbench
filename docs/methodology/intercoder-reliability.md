# Inter-Coder Reliability

The workbench separates raw coder judgments from reconciled case scores.

## Near-Term Plan

- Store raw coder scores separately from reconciled scores.
- Track coder identity, codebook version, definition references, and review status.
- Compute percent agreement for early calibration.

## Later Plan

- Add Krippendorff's alpha or related agreement metrics when independent coding volume is sufficient.
- Preserve adjudication notes separately from raw coder records.

AI-generated coding may be used as draft assistance, but publication-ready claims require human review.
