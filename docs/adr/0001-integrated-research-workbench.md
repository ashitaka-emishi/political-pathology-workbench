# 0001: Integrated Research Workbench

Status: accepted for scaffold

## Context

The project needs to hold theory, evidence, cases, scoring, analysis, and publication in one research operating system.

## Decision

Use a single repository scaffold with data, schemas, JS tooling, Python analysis, Quarto publication, workflows, and documentation.

## Consequences

The repository is polyglot, but the end-to-end workflow is visible and testable from the start.
